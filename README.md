# Hackathon Entry Analyzer

MVP tool to scrape, categorize, and search hackathon entries.

## Quick Start

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Scrape entries:
```bash
python scraper.py
```

3. Launch search interface:
```bash
streamlit run search_app.py
```
